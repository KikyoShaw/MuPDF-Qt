# MuPDF-Qt
基于MuPDF开发的简易PDF阅读器

由于MuPDF用到的库是64位的
开发环境Qt5.12.11+VS2017（x64）

1.支持PDF查看

2.支持上一页，下一页，跳转功能

3.支持放大缩小功能

4.支持PDF打印功能
